*Deployment after making changes to files* 

*Stage all changes*

git add .

*Commit your changes*

git commit -m "Your commit message"

*Push changes to your GitHub repository*

git push origin main

*Build and deploy the changes to GitHub Pages*

npm run deploy

*Branch should be gh-pages*
